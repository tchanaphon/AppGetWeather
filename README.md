# AppGetWeather

for build project\
Gradle Plugin Version 4.2.2\
Gradle Version 7.5.1

support minSDK level 24

get Weather from API. query by name of city. and weather forecast every 3 hour.

- MVVM Pattern
- Coroutine Flow
- LiveData
- Dagger (Hilt)
- Navigation
- CoordinatorLayout
